## 🎮 게임 조작법

* **마우스 좌클릭 / Space / Enter:** 대화창 넘기기 및 선택지 클릭 조작
* **타이틀 화면 전용 버튼:**
  * **`+` / `-` 버튼:** BGM 배경음악 볼륨을 10% 단위로 조절합니다.
  * **`❓ 게임 도움말` 버튼:** 클릭 시 조작법과 시스템 핵심 요약이 담긴 반투명 모달 팝업창을 출력합니다.
* **ESC 키:** 게임을 즉시 안전하게 종료합니다.

## 📊 시스템 기능 소개

### 1. 4대 핵심 스탯 시스템

* **학점 (0 ~ 100):** 시험과 과제 결과에 영향을 받으며, 최종 학점 등급(A+ ~ F)을 결정합니다.
* **멘탈 (0 ~ 100):** 주차가 넘어갈 때마다 피로도 디케이(-5)가 발생합니다. 선택지에 따라 회복 및 깎임이 일어납니다.
* **인간관계 (0 ~ 100):** 동기들과의 의리 지수입니다. 특정 수치 이상일 때 주차별 '친구 특별 이벤트'가 발동합니다.
* **소지금 (실제 화폐 단위):** 게임 내부 스탯 10당 10,000원의 현실적인 현금 단위로 변환되어 표기됩니다. 야식 결제, 스타일링 쇼핑, 단기 알바 보상에 연동됩니다.

### 2. 고급 UI/UX 기능 (AI 고도화 요소)

* **🎯 스탯 크리티컬 경고 연출:** 학점, 멘탈, 인간관계 중 하나라도 20 이하로 떨어지면, HUD 텍스트와 게이지 바가 붉은색으로 긴박하게 깜빡이며 게임오버 위기감을 고조시킵니다.
* **📈 실시간 우측 정렬 플로팅 텍스트:** 선택지를 누르면 어떤 스탯이 오르고 내렸는지(예: 소지금 -25,000원, 학점 +15) HUD 게이지 바로 왼쪽에 명확한 한글로 표기되며, 2.5초(150프레임) 동안 자연스럽게 이동하다 사라집니다.
* **👁️ 실시간 배경 명도 분석 알고리즘:** 배경 이미지의 명도를 실시간으로 추적하여 배경이 밝으면 UI 글씨를 어두운 톤으로, 배경이 어두우면 흰색 파스텔톤으로 카멜레온처럼 가변 변환하여 완벽한 가독성을 보장합니다.
* **🎲 확률적 무작위 분기 시스템:** AI 과제 대행, 교수님 어깨동무 처세술 등 도박성 선택지는 50%의 무작위 확률에 따라 대성공 혹은 대실패 씬으로 분기됩니다.
