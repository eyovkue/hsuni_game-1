# 🎓 한성대학교 신입생 생존기

> **1학년 1학기를 살아남아라!**  
> 수강신청부터 기말고사까지, 선택이 당신의 대학 생활을 결정합니다.

---

## 📌 프로젝트 소개

한성대학교 신입생이 되어 1학년 1학기를 헤쳐나가는 **비주얼 노벨 게임**입니다.  
매 주차마다 현실적인 대학 생활 선택지가 주어지며, 플레이어의 결정에 따라  
**학점 · 멘탈 · 인간관계 · 소지금** 스탯이 변화합니다.

## 📁 폴더 구조

```
HANSUNGFRESHMANSURVIVAL/
├─ main.py              ← 게임 엔진 (Pygame)
├─ player.py            ← 플레이어 스탯 클래스
├─ story.py             ← 전체 씬/대사/선택지 데이터
├─ prompt_history.md    ← AI 프롬프트 기록
├─ README.md            ← 이 파일
└─ assets/
   ├─ backgrounds/      ← 배경 이미지 (PNG/JPG)
   ├─ characters/       ← 캐릭터 스프라이트 (PNG)
   └─ audio/            ← BGM / 효과음 (OGG/WAV)
```

## ⚙️ 설치 및 실행

### 1. 요구사항

- Python 3.8 이상
- [Pygame](https://www.pygame.org/) 라이브러리

### 2. 설치

```bash
pip install pygame
```

### 3. 실행

```bash
cd HANSUNGFRESHMANSURVIVAL
python main.py
```

## 🎮 게임 조작법

| 키 / 마우스 | 동작 |
|:-----------:|:----:|
| `클릭` / `Space` | 대사 넘기기 / 텍스트 스킵 |
| `클릭` | 선택지 선택 |
| `ESC` | 게임 종료 |

## 📊 스탯 시스템

| 스탯 | 설명 | 범위 |
|:----:|:----:|:----:|
| 학점 | 수업·과제 성실도 | 0 ~ 100 |
| 멘탈 | 심리적 여유 | 0 ~ 100 |
| 인간관계 | 친구·선배와의 친밀도 | 0 ~ 100 |
| 소지금 | 보유 금액 (단위: 만원) | 0 ~ 100 |

### 최종 학점 등급
| 점수 | 등급 |
|:----:|:----:|
| 90+ | A+ |
| 80+ | A  |
| 70+ | B+ |
| 60+ | B  |
| 50+ | C+ |
| ~49 | C  |
